# Mandatory Antigravity Verification & Git Rule

> [!CRITICAL]
> **Mandatory Verification Before Git Operations**:
> Before executing any `git commit` or `git push` command, the agent MUST ALWAYS:
> 1. Run type checking (`npx tsc --noEmit` or `npm run typecheck`) and confirm 0 errors.
> 2. Run automated test suites (`npm run test` or `vitest run`) and confirm all tests pass cleanly.
> 3. Resolve all compilation, linting, or type errors BEFORE staging and pushing changes.
> 4. NEVER push unverified code to git repositories or deployment branches.
